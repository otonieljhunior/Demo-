Projeto Spring AI – Sistema de Cadastro com MySQL

Este projeto consiste em uma aplicação Spring Boot com integração ao Spring AI, utilizando MySQL como banco de dados.
O sistema implementa:

Cadastro de três entidades básicas: Cliente, Produto e Pedido.

Associação obrigatória: Pedido → Cliente (um cliente possui vários pedidos).

Telas de cadastro, consulta com filtro, edição (update) e exclusão (delete).
